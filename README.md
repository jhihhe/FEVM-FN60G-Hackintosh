## ⚠️ 重要提示  
**必须自行生成以下参数并修改对应配置**：  
- Board Serial Number  
- 序列号  
- SmUUID  
- SysParameter 系统参数中的"自定义UUID"  
- RtVariables 变量设置中的 MLB、ROM
